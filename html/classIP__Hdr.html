Detailed Description

A class that wraps either an IPv4 or IPv6 packet and abstracts methods for inquiring about common features between the two.

Construct the header wrapper from an IPv6 packet. Caller must have already checked that the static IPv6 header is not truncated. If the packet contains extension headers and they are truncated, that can be checked afterwards by comparing len with TotalLen. E.g. NetSessions::DoNextPacket does this to skip truncated packets.
