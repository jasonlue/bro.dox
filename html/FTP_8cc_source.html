218  // that includes an identifier for the mechanism type. The
219  // framing is supposed to be required for the initial context
220  // token, but GSI doesn't do that and starts right in on a
221  // TLS/SSL handshake, so look for that to identify it.

226  // record from the first byte (content type of 0x16) and
