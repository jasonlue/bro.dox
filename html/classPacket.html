Detailed Description

A link-layer packet.

Note that for serialization we don't use much of the support provided by the serialization framework. Serialize/Unserialize do all the work by themselves. In particular, Packets aren't derived from SerialObj. They are completely seperate and self-contained entities, and we don't need any of the sophisticated features like object caching.
