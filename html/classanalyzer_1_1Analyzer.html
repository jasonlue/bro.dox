Detailed Description

Main analyzer interface.

Each analyzer is part of a tree, having a parent analyzer and an arbitrary number of child analyzers. Each analyzer also has a list of SupportAnalyzer. All analyzer input first passes through this list of support analyzers, which can perform arbitrary preprocessing.

When overiding any of the class' methods, always make sure to call the base-class version first.
