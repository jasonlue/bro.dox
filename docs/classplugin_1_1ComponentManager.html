Detailed Description

template<class T, class C>
class plugin::ComponentManager< T, C >

A class that manages tracking of plugin components (e.g. analyzers) and installs identifiers in the script-layer to identify them by a unique tag, (a script-layer enum value).

Template Parameters
TA Tag type or derivative.
CA plugin::TaggedComponent type derivative.
