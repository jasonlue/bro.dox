virtual double iosource::IOSource::NextTimestamp ( double *  network_time)
pure virtual

Returns the timestamp (in global network time) associated with next data item from this source. If the source wants the data item to be processed with a local network time, it sets the argument accordingly.

This method will be called only when either IsIdle() returns false, or select() on one of the fds returned by GetFDs() indicates that there's data to process.

Must be overridden by derived classes.

Parameters
network_timeA pointer to store the local network time associated with the next item (as opposed to global network time).
Returns
The global network time of the next entry, or a value smaller than zero if none is available currently.

Implemented in iosource::PktSrc, bro_broker::Manager, EventPlayer, RemoteSerializer, DNS_Mgr, and threading::Manager.
