Constructs a prefix instance from an IPAddr object and prefix length.

Parameters
addrThe IP address.
lengthThe prefix length in the range from 0 to 128
len_is_v6_relativeWhether length is relative to the full 128 bits of an IPv6 address. If false and addr is an IPv4 address, then length is expected to range from 0 to 32. If true length is expected to range from 0 to 128 even if addr is IPv4, meaning that the mask is to apply to the IPv4-mapped-IPv6 representation.
