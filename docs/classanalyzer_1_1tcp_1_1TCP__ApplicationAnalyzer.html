void TCP_ApplicationAnalyzer::ProtocolViolation ( const char *  reason,
const char *  data = 0,
int  len = 0 
)
overridevirtual

Signals Bro's protocol detection that the analyzer has found a severe protocol violation that could indicate that it's not parsing the expected protocol. This turns into protocol_violation events at the script-layer (one such event is raised for each call to this method so that the script-layer can built up a notion of how prevalent protocol violations are; the more, the less likely it's the right protocol).

Parameters
reasonA textual description of the error encountered.
dataAn optional pointer to the malformed data.
lenIf data is given, the length of it.
